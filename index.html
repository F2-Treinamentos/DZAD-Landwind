O que você vai aprender neste curso?

  • Fundamentos de programação: Introdução aos conceitos básicos de programação, como variáveis, estruturas de controle e funções.
  • Desenvolvimento web: Criação de sites e aplicações web utilizando HTML, CSS e JavaScript. Aprender a estruturar páginas, estilizá-las e adicionar interatividade.
  • Banco de dados: Introdução aos conceitos de bancos de dados e aprendizado de SQL para manipulação e consulta de dados.
  • Frameworks e bibliotecas: Exploração de frameworks populares como Laravel, Livewire, Alpine JS, Filament e Vue.js para o desenvolvimento de aplicações web mais complexas.
  • Desenvolvimento de APIs: Aprender a criar e consumir APIs (Application Programming Interfaces) para integração de sistemas.
  • Gerenciamento de versão: Utilização de ferramentas como Git para controle de versão do código fonte.
  • Implantação (deploy): Entender como implantar uma aplicação web em um servidor ou plataforma de hospedagem.
